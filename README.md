ABOUT Cuddi 

### How do I get set up? ###

 - Clone the GIT repository
 - Checkout into the latest git branch
 - Install Node package dependencies using  ``npm i``
 - Develop or modify changes
 - Build and run it into the browser using ``npm start`` command
